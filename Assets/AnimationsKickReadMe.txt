---------------------------------------------------------------
INSTALLATION AND USAGE
---------------------------------------------------------------
Import the package into your project.
The folder "Animations" contains 5 animations.
Folder "FromStealth" contains the 3D model from the unity example project.
The folder "Scene" contains a scene with an animated character.

---------------------------------------------------------------
ANIMATION LIST
---------------------------------------------------------------
- Axe kick
- Back kick
- Brazilian kick
- Defensive side kick
- Front kick
